# Project Overview

NeuroAI is a modern and fully responsive AI Startup Landing Page developed using **HTML5** and **Tailwind CSS**. The project demonstrates the use of Tailwind utility classes to build a clean, professional, and responsive user interface without using custom CSS.

The website is designed with a modern SaaS (Software as a Service) style and includes multiple reusable UI components.

# How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Ensure all images are placed inside the **assets** folder.
4. Open **index.html** in any modern web browser.
5. Internet connection is required for Tailwind CSS, Google Fonts, and Font Awesome CDN.
